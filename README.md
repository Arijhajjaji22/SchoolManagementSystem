# School Management System

## Description
Web application to manage school operations, including students, teachers, administrators, and other staff.  
Provides an interface for different user roles to access and manage relevant data.

## Features
- **Student Management**: Register, edit, and track student information.
- **Teacher Management**: Manage teacher profiles and classes.
- **Admin & Director Dashboard**: Oversee school activities, reports, and users.
- **Employee Management**: Manage other staff members.
- **Role-based Access**: Different views and permissions for students, teachers, admins, directors, and other staff.

## Tech Stack
- Backend: .NET 8, C#
- Frontend: Razor, HTML5, CSS3, JavaScript
- Database: SQL Server / MySQL
- DevOps (if applicable): Docker, Jenkins
- Version Control: Git
